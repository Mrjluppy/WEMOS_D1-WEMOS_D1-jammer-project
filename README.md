## âš ï¸ Battery Safety Notice
Li-ion / Li-Po batteries are small but **require proper handling**:

- Only charge using a **protected TP4056 module** (with protection circuit).  
- Never connect a Li-Po battery directly to the board without protection.  
- Stop using immediately if the battery becomes **hot, swollen, or smells unusual**.  
- Do not short-circuit, puncture, bend, or expose the battery to heat.  
- Keep inside a protective case and do not leave charging unattended.  

> If you are not experienced with Li-Po batteries, powering the project via **USB** is the safest option.
